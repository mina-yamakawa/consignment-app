# 委託販売管理システム

Flaskで作成した委託販売管理Webアプリです。  
売上管理・在庫管理・期限管理・月次集計を行えます。

## URL

http://slow-0622-app.com

## 使用技術

- Python / Flask
- SQLite
- HTML / CSS
- AWS EC2
- nginx
- gunicorn
- systemd
- お名前.com DNS

## 機能

- ログイン機能
- 売上登録
- 在庫管理
- 賞味期限管理
- 月別売上集計
- 利益計算

## インフラ構成

EC2 + nginx + gunicorn + Flask

## 制作目的

家業の委託販売事業の管理を効率化するために作成しました。
